How do I get setup for workshops?

To avoid mis-copying the installation instructions, please follow the instructions that Docker has outlined. (They're linked in the headings. If any links are broken, please inform the coordinators about which are.)

  • NOTE: We apologize for the inconvenience, but until Docker expands it's support for Windows 10 beyond the Pro, Education, and Enterprise editions; we won't support Windows.
  • NOTE: Docker only supports 64-bit systems across all OSes.

    2. Installing docker-compose. Follow the steps listed here.

2. Getting the Notebook Running (for macOS and Linux):

  1. Open your Terminal.
  2. Execute: git clone https://github.com/ucfsigai/<workshop_name>.git.
  3. Execute: cd /path/to/cloned/repo.
  4. Execute: docker-compose up.
  5. Open your browser and go to: http://localhost:19972/.
